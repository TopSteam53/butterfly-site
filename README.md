# Butterfly — сайт студии лазерной эпиляции

Готовая первая версия на Next.js без дополнительных UI-библиотек.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Что уже работает

- адаптивная desktop/mobile вёрстка;
- все основные секции;
- реальные пакеты и цены из VK;
- калькулятор подбора пакета;
- FAQ-аккордеон;
- кнопки Dikidi, VK и телефона;
- закреплённая мобильная кнопка записи;
- SEO metadata;
- подпись Makaroff Studio.

## Что заменить перед запуском

1. В `app/layout.js` заменить `https://example.com` на настоящий домен.
2. Добавить реальные фотографии мастера, кабинета и оборудования.
3. Заменить демонстрационные отзывы.
4. Подтвердить актуальность скидки −20%.
5. Встроить настоящую Яндекс Карту.
6. Подключить Яндекс Метрику и цели.

## Деплой на Vercel

1. Загрузите проект в GitHub.
2. В Vercel выберите **Add New → Project**.
3. Подключите репозиторий.
4. Нажмите **Deploy**.
